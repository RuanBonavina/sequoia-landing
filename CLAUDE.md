## Project

**Sequoia Landing** — a landing page project. The repository is in its initial state; technology stack and build tooling have not yet been configured.

Remote: `git@github.com:RuanBonavina/sequoia-landing.git`

## Getting Started

```bash
npm run dev   # starts local server at http://localhost:3000 (via npx serve)
```

No build step needed — the project uses Tailwind CSS via CDN.

## Stack

- Pure HTML + Tailwind CSS 3 (CDN) + Google Fonts (Noto Serif, Manrope) + Material Symbols
- `index.html` — main landing page (Instituto Sequóia, pt-BR)
- `prd.html` — project PRD template
